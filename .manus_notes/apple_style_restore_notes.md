# Apple Style Restore Notes

## 목표 화면 핵심 요소

| 영역 | 목표 상태 |
|---|---|
| 전체 배경 | 딥 스페이스 블랙 계열 외곽 프레임 안에 실버 메탈 카드가 세로로 정렬된 구조 |
| 상단 카드 | 밝은 네온 에메랄드 GO 카드, 얇은 실버 베젤, 카드 모서리 둥글고 내부 텍스트는 중앙 정렬 |
| 중단 정보 카드 | 실버 카드 한 장 안에 남은 거리와 현재 속도를 2열로 분할, 군더더기 없는 배치 |
| 중앙 HUD | 큰 화살표 1개와 짧은 방향 텍스트만 보이는 단순 카드 |
| 하단 바 | 흰색/실버 계열의 작은 3버튼 바, 카메라·홈·설정이 동일 비중으로 정렬 |

## 현재 화면과의 차이

| 영역 | 현재 상태 | 복구 방향 |
|---|---|---|
| 중앙 HUD | 네이비 블루 카드와 과도한 장식이 보임 | 실버 메탈 카드로 단순화 |
| 하단 바 | 목표보다 높고 존재감이 큼 | 더 작고 얇은 애플식 조작 바로 축소 |
| 전체 톤 | 일부 영역이 지나치게 진하거나 무거움 | 실버와 블랙의 대비는 유지하되 더 정제된 애플 톤으로 정리 |
| 카드 디테일 | 목표보다 두껍거나 깊은 장식 느낌 | 얇은 베젤과 절제된 입체감으로 정리 |
