# 음성 명령어 가이드

AI Omni Drive 앱에서 사용 가능한 음성 명령어 목록입니다.

## 🎯 음성 인식 품질 향상 기능

- **자동 재시도**: 음성이 명확하지 않거나 신뢰도가 낮을 때 자동으로 재시도 (최대 2회)
- **노이즈 제거**: 에코 캔슬링, 노이즈 억제, 자동 게인 컨트롤 적용
- **다중 프레임 합의**: 여러 번의 인식 결과를 비교하여 정확도 향상
- **컨텍스트 기반 보정**: 현재 상황에 맞춰 명령어 자동 해석
- **통계 추적**: 음성 인식 성공률 및 평균 신뢰도 실시간 표시

## 💡 사용 팁

1. **명확하게 발음하기**: 또렷하게 말하면 인식률이 높아집니다.

2. **짧고 간결하게**: "신호등 상태" 처럼 핵심 단어만 말해도 인식됩니다.

3. **재시도 자동화**: 인식이 실패하면 자동으로 재시도하므로 기다려 주세요.

4. **컨텍스트 활용**: 
   - 내비게이션 중: "몇 미터" → 자동으로 "신호등 몇 미터"로 해석
   - 음악 재생 중: "다음" → 자동으로 "다음 곡"으로 해석

5. **통계 확인**: 음성 명령 후 화면 상단에 인식 통계가 표시됩니다.

## 🔧 문제 해결

### 음성이 인식되지 않을 때
- 주변 소음을 줄이고 다시 시도해 보세요
- 마이크에 가까이 대고 말해 보세요
- 브라우저 마이크 권한을 확인해 보세요 (웹 버전)

### 잘못 인식될 때
- 천천히 또렷하게 발음해 보세요
- 핵심 단어 위주로 짧게 말해 보세요
- 도움말 명령어로 정확한 표현을 확인해 보세요

### 자주 실패할 때
- "스캔 통계" 명령어로 현재 인식률을 확인해 보세요
- 저시력 모드를 켜면 시각적 피드백이 커집니다
- 설정에서 음성 안내 볼륨을 조절해 보세요
